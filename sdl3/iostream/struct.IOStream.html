pub fn from_read<T>( r: &mut T, buffer: &'a mut Vec<u8>, ) -> Result<IOStream<'a>, Error>
where T: Read + Sized,

Reads a Read object into a buffer and then passes it to IOStream.from_bytes.

The buffer must be provided to this function and must live as long as the IOStream, but the IOStream does not take ownership of it.
